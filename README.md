# Digital image processing using opencv and ubidots iot platform
IoT People Counter using DIP , OpenCV, and Ubidots.

1.Download open-cv in your python environment and other dependencies.

2.Create your ubidots account and put the token which is received by you. Then use some device name and variable name for your application.

3. put some sample images in your path.

4.To analyze these images, first your must store the images on your laptop or PC and track the path to analyze the images. Run the peoplecount python code from cmd.

python peopleCounter.py -i dataset/image_1.png
